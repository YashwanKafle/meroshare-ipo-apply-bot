# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.** Instead, email the maintainer directly:

**Email:** yashwankafle@gmail.com

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact

You can expect an acknowledgment within **48 hours** and a resolution timeline within **7 days**.

## Security Best Practices for Users

- **Never commit your `accounts.csv` file** to version control. Add it to `.gitignore`.
- Keep your MeroShare credentials secure and do not share them.
- Use this tool only on trusted devices.
